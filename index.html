API Developer 1

A software developer who can interact with simple REST API's.

Master the following skills

  1. Understand the utility of REST API's
  2. Able to read and write to a REST API using a REST Utility
  3. Understands how and when to use querystring
  4. Can use API Keys to gain access to secured data
  5. Read API responses.
  6. Properly form JSON payloads for requests
  7. Recognize common HTTP Error codes (200's, 300's, 400's, 500's)
  8. Utilize JavaScript's built-in Fetch (get, put, post, delete) to interact with an API
  9. Utilize JQuery's AJAX functions (get, put, post, delete) to interact with an API
  10. Process API response to display data on a web page
